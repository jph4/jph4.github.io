JP Hilton

JP Hilton Collection includes source code for static websites, WordPress, ReactJS, WooCommerce, Bootstrap, ASP, and jQuery, as well as many other frameworks (angular, ...). JPH (JP Hilton) was written originally to support the JPH web sites . The JPH web site was developed to be 100% efficient, efficient in the sense that it respects the user's needs.

We strive to provide regular, high quality releases, which we want to work well on a variety of native and cross targets, and encourage everyone to constribute changes or help testing JPH.
